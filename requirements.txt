# Route Note PWA

Application web progressive pour enregistrer des livraisons avec :
- mode manuel
- mode départ / arrêt automatique
- géolocalisation GPS via l'appareil
- stockage local dans le navigateur
- installation PWA

## Fichiers
- `index.html` : interface
- `app.js` : logique applicative
- `manifest.webmanifest` : configuration PWA
- `sw.js` : service worker
- `icons/` : icônes PWA
- `server.py` : petit serveur local HTTPS pour tests GPS/PWA
- `requirements.txt` : dépendances Python pour le serveur

## Lancer localement
### Option simple HTTP
```bash
python3 -m http.server 8080
```

Puis ouvrir :
- `http://localhost:8080`

### Option HTTPS recommandée pour géolocalisation
```bash
pip install -r requirements.txt
python3 server.py
```

Puis ouvrir :
- `https://localhost:4443`

Le navigateur peut demander d'accepter le certificat local auto-signé.

## Déploiement
Tu peux mettre ce dossier sur :
- un hébergement web HTTPS
- GitHub Pages
- Netlify
- Vercel
- ton propre serveur Nginx / Apache

## Notes importantes
- Le GPS donne une **distance estimée**, pas le compteur réel du véhicule.
- Le stockage est en **localStorage**, donc les données restent sur l'appareil/navigateur.
- Pour un vrai backend API multi-appareils, il faudra ensuite ajouter une base de données et une API serveur.

## API utilisée
L'application utilise l'API navigateur :
- `navigator.geolocation.watchPosition()`

Il n'y a pas d'API externe obligatoire dans cette version.
